Description

Library to parse json files and convert, them into armadillo adjacency matrix

WARNING json file must be something like:

{
"master_key":
[
{
"node_key": "node1",
"nn_key": ["node3", "..."]
},
{
"node_key": "node3",
"nn_key": []
},
{
...
...
},
]
}

So at master_key a list of dict, each with a node name (type: string) at node_key and its first neighbors nodes (type: list string) at nn_key

Also, all nn_nodes themselves must appear as nodes, at most they will not present any links (dangling nodes) but they must be present to make the graph consistent
